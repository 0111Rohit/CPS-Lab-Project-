# Projects
📌Project Overview

The CPS Lab Website is a fully functional web platform developed for the university’s Cyber-Physical Systems (CPS) Lab.
The lab is powered by Indian Institute of Technology Ropar and co-powered by AWaDH.

This website serves as an official digital platform to showcase lab activities, research projects, events, faculty details, and student contributions.
